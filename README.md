# OpenIFEM
An implementation of the Immersed Finite Element Method based on deal.II

## Dependencies
1. MPICH/Open MPI
2. PETSc with `MUMPS` and `Hypre`
3. p4est
4. metis
5. deal.II greater than v9.1 with 1-4
